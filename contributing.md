# Contributing to Mochi

Mochi is free and open source software, developed by a small team. The source is always available and anyone is free to fork it. A feature described in words is usually faster and more reliable for us to implement than a code contribution is to review, so feature requests and bug reports are what we usually prefer. Good ones are implemented quickly.

We will consider code contributions, but only from people who have signed the Mochi Contributor Agreement, which covers copyright and patents. If you would like to contribute code, please discuss it with us before starting work, so we can agree the approach and get the agreement signed first. Please do not send unsolicited pull requests, or post code, patches, or snippets in issues. We cannot use code from anyone who has not signed the agreement, so to keep the copyright clean we will not read it, and unsolicited or unsigned contributions are closed or left unread.

Contact: **developers@mochi-os.org**
