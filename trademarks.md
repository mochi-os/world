- Use any Mochi Mark in connection with software or services unrelated to Mochi.

## Hosting and services

You may state truthfully that you offer "Mochi hosting" or "managed Mochi" where the software you run is unmodified Mochi. If you run a modified version, describe it as your own product derived from Mochi, not as Mochi.

If you distribute a modified version, we ask that you acknowledge its derivation - for example, "Derived from Mochi (mochi-os.org)" in your documentation or about screen. This is a request, not a condition of the software licence.
